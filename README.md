# Apartment Management System

A full-stack Apartment Management System built with a Spring Boot backend, MySQL database, and a simple HTML/CSS/JavaScript frontend. The application helps manage buildings, apartment units, tenants, lease agreements, and maintenance requests through RESTful API endpoints.

## Features

- Manage apartment buildings
- Manage apartment units
- Add, update, and delete tenants
- Track lease agreements
- View active leases
- Create and manage maintenance requests
- Filter maintenance requests by status or priority
- RESTful backend API using Spring Boot
- MySQL database integration
- Simple browser-based frontend

## Tech Stack

### Backend

- Java 17+
- Spring Boot
- Spring Data JPA
- Maven
- MySQL

### Frontend

- HTML
- CSS
- JavaScript

### Database

- MySQL 8.0+
